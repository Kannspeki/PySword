# PySword
A simple python password generator  

## Usage

```./pysword```  
* Choose the password length, any positive integer    
```password lenght: ```  
* Select the type of password, multiple choices supported e.g. ```ns``` for numbers and symbols  
```(n)umbers, (l)etters, (s)ymbol:```  
* Output  
```%*4!5|!```
